# Student Management System

## Description

The **Student Management System** is a comprehensive web-based solution designed to streamline the management of student information within an educational institution.  

This application simplifies and automates various student-related tasks such as record management, attendance tracking, and overall data handling.

---

## Project Structure

The project folder contains the following files and directories:

- **Static/**  
  Contains stylesheets, images, and JavaScript files used for the frontend.

- **Template/**  
  Includes all HTML templates used in the application (e.g., landing page, attendance pages, etc.).

- **main.py**  
  The main Python file containing the application logic.

- **students.sql**  
  Database file used to set up the project database.

- **requirements.txt**  
  Contains all required Python libraries for the project.

- **Presentation file**  
  Provides an overview and explanation of the project.
